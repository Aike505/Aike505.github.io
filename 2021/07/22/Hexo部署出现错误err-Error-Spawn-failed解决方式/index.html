Hexo部署出现错误err:Error:Spawn failed解决方式

hexo部署博客bug
  • hexo-bug

问题描述

部署中可能会出现错误

总结一下：

问题大多是因为git进行push或者hexo d的时候改变了一些.deploy_git文件下的内容。

解决办法

  1. 删除.deploy_git文件夹;
  2. 输入git config --global core.autocrlf false
  3. 然后，依次执行：
    hexo clean
    hexo g
    hexo d

问题解决。暴力直接，有效。
